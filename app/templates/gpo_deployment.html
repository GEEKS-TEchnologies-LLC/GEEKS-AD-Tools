Deployment Instructions

  1. Build your Credential Provider DLL in Visual Studio
  2. Copy the DLL to your network share
  3. Generate a deployment script using the form above
  4. In Group Policy Management:
    • Create a new GPO for Credential Provider deployment
    • Add the generated script as a startup script
    • Link the GPO to your target OUs
  5. Run gpupdate /force on target machines
